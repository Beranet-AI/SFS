# Technology Stack

## Backend
- Python 3.11
- Django (Management, ORM, migrations)
- FastAPI (stateless services)

## Frontend
- TypeScript
- Next.js (App Router)

## Data
- PostgreSQL (persistent store)
- In-memory cache (live status)

## DevOps
- Docker / Docker Compose
- GitHub Actions (planned)

## AI / ML
- Python-based models
- Pluggable model interfaces

---

## Rationale

- Django chosen for strong domain modeling and migrations
- FastAPI chosen for performance and simplicity
- Next.js chosen for modern UI and SSR capabilities
