# Glossary (Ubiquitous Language)

| Term | Meaning |
|----|----|
| Livestock | An individual animal under monitoring |
| Device | A physical sensor or gateway producing telemetry |
| Telemetry | Raw sensor data with timestamp |
| LiveStatus | A snapshot of the latest telemetry values |
| Health | Evaluated condition of a livestock over time |
| Incident | A condition requiring attention or action |
| Rule | Deterministic condition applied to telemetry |
| Prediction | AI-generated assessment of future health risk |
| Edge Node | On-farm gateway bridging devices to the cloud |
| Source of Truth | The authoritative owner of a piece of data |

Important distinctions:
- **Telemetry ≠ Health**
- **LiveStatus ≠ Event history**
- **Incident ≠ Alert (alerting is a UI concern)**
