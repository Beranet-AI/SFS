# Product Roadmap

## Phase 1 – Foundation (Current)
- Modular backend architecture (Django + FastAPI)
- Telemetry ingestion and rule-based incidents
- Web dashboard MVP
- Dockerized deployment

## Phase 2 – Intelligence
- AI health prediction service
- Risk scoring and trend analysis
- Model versioning and evaluation

## Phase 3 – Scale & Reliability
- Redis for live status
- Async ingestion pipelines
- Horizontal scaling of FastAPI services

## Phase 4 – Ecosystem
- Mobile application
- External integrations (vet systems, reporting)
- Advanced analytics and benchmarking
